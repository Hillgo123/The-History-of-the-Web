The History of the World Wide Web
1960 â€“ 2010

J.C.R. Licklider of MIT proposes a "Galactic Network" concept of globally interconnected computers, which laid the foundation for the development of the ARPANET.

ARPANET is established, which laid the foundation for the internet as we know it today.

The first email is sent by Ray Tomlinson.

The term "Internet" is first used to describe the global network of interconnected computers.

The Domain Name System (DNS) is introduced, which allowed websites to be named with human-readable domain names instead of IP addresses.

Tim Berners-Lee proposes the World Wide Web while working at CERN, a system of interlinked hypertext documents that could be accessed through the internet.

The first web page is created by Tim Berners-Lee.

The Mosaic web browser is released, making the web more accessible to the general public.

Wikipedia is launched, enabling collaborative editing of online content

The term "Web 2.0" is coined to describe the shift towards more dynamic and user-driven web applications.

Instagram is launched, sparking a new era of social media platforms that prioritize visual content.

The first image of a black hole is published online, highlighting the web's ability to share groundbreaking scientific discoveries with the world.
